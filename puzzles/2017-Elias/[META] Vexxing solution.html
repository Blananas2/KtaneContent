Vexxing — 31.3

Solution

The flavour text contains many words consisting of the letters RATE plus an extra letter, hinting that the feeder answers for this meta are answers that are similarly one letter away from a common set of letters. That common set turns out to be STARE.

The puzzle’s title refers to vexillology, which hints at Semaphore (the Vexillology module did not exist at the time). Encoding each of the extra letters into flag semaphore and using those flag positions to refer to the letter grid provided, a set of letter pairs is obtained. There is only one way to reorder these to obtain a solution word.
